Best practices do work

Some influential people have framed CSS as a flawed technology. They've encouraged CSS authors to brute force layout in ways that don't make the most of CSS's features.

With our introductory chapters, the "rudiments", we catch you up on just how smart and elegant modern CSS can be. What you learn in these chapters is then applied to specially designed, modular layout solutions.

We teach you best practices that are guaranteed to make you a better, well-rounded CSS programmer.
